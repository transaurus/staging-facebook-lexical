@lexical/code

This package contains the functionality for the code blocks and code highlighting for Lexical.

Note that the prism highlighting functionality from this module is deprecated and will be removed in a future version of lexical. If you intend to use code highlighting, depend directly on @lexical/code-shiki or @lexical/code-prism.
